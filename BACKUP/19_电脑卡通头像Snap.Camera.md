# [电脑卡通头像Snap Camera](https://github.com/phh95/gitblog/issues/19)

Snap Camera，是 Snapchat 推出的客户端软件，支持 Windows 和 macOS 系统。    

它可以制作类似下图的卡通头像，将人脸替换成看起来好好捏的卡通头像，像是我们在迪士尼动画中看到的人物。    

![image](https://user-images.githubusercontent.com/29042617/145678510-3f91852a-62dd-4a4c-b61d-2a1a4abeab63.png)     

下载地址：   

官网 https://snapcamera.snapchat.com/download/       

官网无法顺利下载，可以移步我的公众号「效率工具指南」，在后台回复「snap」，可获取 Windows 和 macOS 版本。    

Snap Camera 可以配合其他软件一起使用，例如 OBS、Zoom、腾讯会议等。
